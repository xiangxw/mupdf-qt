3 ## Compile on Linux
4 #### Install tools and thirdparty libraries
5 You should install some tools and thirdparty libraries. For Ubuntu users, you can use the following command(similar for other Linux users):
6 
7  sudo apt-get install build-essential pkg-config cmake qtbase5-dev qt5-default libfreetype6-dev

10 
11  git clone https://github.com/xiangxw/mupdf-qt.git
12  cd mupdf-qt
13  git submodule update --init --recursive
14

22  cd ..

42 ## Compile on Window with MSYS MinGW
43 #### Clone source code
44 
45  git clone https://github.com/xiangxw/mupdf-qt.git
46  cd mupdf-qt
47  git submodule update --init --recursive
48 
49 #### Installing MSYS/MinGW
50 Install MSYS and MinGW (http://www.mingw.org/wiki/Getting_Started). Using the Graphical User Interface Installer, [mingw-get-setup.exe](http://sourceforge.net/projects/mingw/files/Installer/mingw-get-setup.exe/download), is recommended. During installation, "checking/ticking" the following in "Basic Setup" is recommended:
51 
52 1. mingw32-base
53 2. mingw32-gcc-g++
54 3. msys-base
55 4. mingw-developer-toolkit
56 
57 Add C:\MinGW\bin; to your PATH system variables (at the beginning).
58 
59 #### Compile MuPDF source
60 
61  cd mupdf-qt
62  make build=debug NOX11=yes
63 
64 For release, just change "debug" to "release". NOX11 is necessary since X11 headers are not available in Windows. Note that this will not build the "app" packaged with the MuPDF source.
65 
66 #### Compile mupdf-qt
67 
68 Install CMake.
69 
70 Open CMakeLists.txt with Qt Creator(Qt5 built with mingw), run cmake and build mupdf-qt
